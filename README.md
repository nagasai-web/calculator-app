1. Project Title
   # calculator-app
2. Description
Created a simple calculator using HTML, CSS, and JavaScript. Includes basic operations like addition, subtraction, multiplication, and division. UI styled using CSS grid layout.

3. Features
-Basic arithmetic operations (+, -, *, /)
- Clear (AC) and delete (DEL) functionality
- Responsive button layout using CSS Grid
- Simple and clean UI

4. Technologies Used
     HTML
     CSS
     JavaScript

5. How to Run
   1. Download or clone the repository
   2. Open `index.html` in your browser
